# PipTracker (Spring Boot + springdoc) - Quick Run

## Build
mvn clean package

## Run
java -jar target/piptracker-0.0.1-SNAPSHOT.jar

## Dev access
Open: http://localhost:8080/v3/portal/ess/home
(This will redirect to the Swagger UI which loads the OpenAPI JSON from /v3/api-docs.)

## Swagger JSON
http://localhost:8080/v3/api-docs
